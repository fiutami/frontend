# Agent Launcher

Invoca l'agente richiesto con: `/agent <nome-agente>`

## Agenti Disponibili:

| Comando | Agente | Descrizione |
|---------|--------|-------------|
| `/agent orchestrator` | Orchestrator | Coordinatore centrale dei task paralleli |
| `/agent frontend` | FrontEndArchitect | Architettura frontend e responsive design |
| `/agent backend` | Backend | API .NET e architettura backend |
| `/agent db` | DBMagician | Database design e ottimizzazione query |
| `/agent api` | APIMaster | Design e implementazione REST/GraphQL API |
| `/agent security` | SecurityGuardian | Sicurezza, autenticazione e vulnerabilità |
| `/agent test` | TestMaestro | Testing e quality assurance |
| `/agent devops` | DevOpsArchitect | CI/CD, Docker e infrastructure |
| `/agent perf` | PerformanceOptimizer | Ottimizzazione performance |
| `/agent docs` | DocArchitect | Documentazione tecnica |
| `/agent figma` | Figma | Integrazione Figma-to-Code |
| `/agent i18n` | I18N | Internazionalizzazione e localizzazione |
| `/agent design` | Design | UI/UX design system |

## Uso:

```
/agent frontend   → Avvia l'agente FrontEndArchitect
/agent backend    → Avvia l'agente Backend
/agent security   → Avvia l'agente SecurityGuardian
```

Specifica quale agente vuoi invocare con il tuo task: $ARGUMENTS
