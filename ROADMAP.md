# Fiutami Frontend Roadmap

## Current Features

- User authentication (email + Google OAuth)
- Pet registration flow
- Species questionnaire
- User profile, settings, security pages
- Responsive design (mobile-first)
- PWA support
- Home Dashboard con pet card
- Drawer menu completo
- Mappa POI con Leaflet
- Chat system (polling)
- Ricerca globale
- Premium page
- Dark mode
- i18n 6 lingue
