## Code Example: Using Translations

### Template (HTML)
```html
<div class="login-page">
  <!-- Title -->
  <h1>{{ 'auth.loginTitle' | translate }}</h1>

  <!-- Form -->
  <form [formGroup]="loginForm">
    <!-- Email Input -->
    <label>{{ 'auth.email' | translate }}</label>
    <input
      type="email"
      formControlName="email"
      [placeholder]="'auth.emailPlaceholder' | translate">

    <!-- Password Input -->
    <label>{{ 'auth.password' | translate }}</label>
    <input
      type="password"
      formControlName="password"
      [placeholder]="'auth.passwordPlaceholder' | translate">

    <!-- Forgot Password Link -->
    <a routerLink="/forgot-password">
      {{ 'auth.forgotPassword' | translate }}
    </a>

    <!-- Submit Button -->
    <button type="submit">
      {{ 'auth.loginButton' | translate }}
    </button>
  </form>

  <!-- Terms -->
  <p class="terms">
    {{ 'auth.termsAccept' | translate }}
  </p>
</div>
```

### Component (TypeScript)
```typescript
import { Component } from '@angular/core';
import { LanguageService, SupportedLanguage } from './core/i18n';

export class LoginComponent {
  currentLanguage: SupportedLanguage;

  constructor(private languageService: LanguageService) {
    this.currentLanguage = this.languageService.currentLanguage;
  }

  switchToEnglish() {
    this.languageService.setLanguage('en');
  }

  getCurrentLanguageName(): string {
    return this.languageService.currentLanguageConfig?.nativeName || '';
  }
}
```

## Testing Different Languages

### Browser Console Commands
```javascript
// Check current language
localStorage.getItem('fiutami_lang')
// Output: "it" (or "en", "fr", "de", "es")

// Switch to English
localStorage.setItem('fiutami_lang', 'en')
location.reload()

// Switch to French
localStorage.setItem('fiutami_lang', 'fr')
location.reload()

// Switch to German
localStorage.setItem('fiutami_lang', 'de')
location.reload()

// Switch to Spanish
localStorage.setItem('fiutami_lang', 'es')
location.reload()

// Back to Italian (default)
localStorage.setItem('fiutami_lang', 'it')
location.reload()
```

### Angular Component
```typescript
// Inject LanguageService
constructor(private lang: LanguageService) {}

// Switch languages programmatically
this.lang.setLanguage('en');  // English
this.lang.setLanguage('it');  // Italian
this.lang.setLanguage('fr');  // French
this.lang.setLanguage('de');  // German
this.lang.setLanguage('es');  // Spanish

// Get current language
console.log(this.lang.currentLanguage);  // "it" | "en" | "fr" | "de" | "es"

// Get instant translation
const text = this.lang.getInstantTranslation('auth.login');
console.log(text);  // "Accedi" (if Italian)
```

## Character Length Comparison

**Important for UI Design:** Text length varies significantly across languages.

| Key | IT (chars) | EN (chars) | FR (chars) | DE (chars) | ES (chars) | Max |
|-----|------------|------------|------------|------------|------------|-----|
| `auth.login` | 6 | 5 | 9 | 8 | 15 | 15 (ES) |
| `auth.forgotPassword` | 25 | 16 | 22 | 20 | 25 | 25 (IT/ES) |
| `auth.termsAccept` | 53 | 50 | 76 | 63 | 58 | 76 (FR) |
| `hero.tagline` | 31 | 30 | 32 | 32 | 28 | 32 (FR/DE) |

**Design Recommendation:** Allow 20-30% extra space for German and French translations, which tend to be longer.
