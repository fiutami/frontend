## Token Update Workflow

1. **Figma Update**: Designer updates design in Figma
2. **Token Sync**: Run `npm run figma:sync-tokens` (when available)
3. **Manual Update**: Edit `src/styles/_tokens-figma.scss` manually
4. **CSS Vars**: Update `src/styles/_tokens.scss` if needed
5. **Mixins**: Add/update mixins in `src/styles/_mixins.scss`
6. **Test**: Run `npm run build` to verify SCSS compilation
7. **Commit**: Commit changes with message: `feat(tokens): update design tokens from Figma LOGIN-STATIC`
